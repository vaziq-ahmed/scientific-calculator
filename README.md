# 🧮 Scientific Calculator

A fully functional, premium dark-themed **Scientific Calculator Web Application** built using **HTML, CSS, and Vanilla JavaScript** — without external frameworks.

This project focuses on clean UI design, modular logic, safe computation handling, and a responsive user experience.

## 📌 Features

### ✅ Basic Arithmetic
- Addition (+)
- Subtraction (−)
- Multiplication (×)
- Division (÷)
- Modulus (%)
- Power (xʸ)

### ✅ Scientific Functions
- sin(x)
- cos(x)
- tan(x)
- log(x) (Base 10)
- ln(x) (Natural log)
- √ (Square root)
- x² (Square)
- π (Pi constant)
- e (Euler’s constant)

### ✅ Advanced Capabilities
- Degree / Radian toggle (DEG / RAD mode)
- Parentheses support
- Memory functions:
  - MC (Memory Clear)
  - M+ (Memory Add)
  - M− (Memory Subtract)
  - MR (Memory Recall)
- Expression history panel
- Copy result to clipboard
- Keyboard input support
- Responsive design (mobile + desktop)

## ⚙️ How to Run Locally

1. Clone the repository:
git clone https://github.com/vaziq-ahmed/scientific-calculator.git


2. Navigate to the project folder:
cd scientific-calculator


3. Open `index.html` in your browser.

No installation or build steps required.

---

## 🌍 Deployment (GitHub Pages)

To deploy:

1. Go to Repository Settings  
2. Navigate to "Pages"  
3. Select branch: `main`  
4. Save  

Your app will be live.
